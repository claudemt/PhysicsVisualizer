Elastic Film GUI (English, grouped notes version)
=================================================

Files
-----
1. main_elastic_film_gui.m
   - Main GUI entry
   - Export logic
   - Build entry

2. elastic_film_formula.m
   - Formula / solver only

How to run
----------
main_elastic_film_gui
or
main_elastic_film_gui('run')

How to build
------------
main_elastic_film_gui('build')

Notes layout
------------
- The GUI uses one grouped Notes panel.
- The Notes panel updates automatically when you switch tabs.
- This avoids repeating note text under every input field.
